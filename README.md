 Java JDBC Banking System

  A console-based Java application that simulates core banking operations such as account validation, balance enquiry, and fund transfer using **JDBC with Oracle Database**.

Project Overview

  This project demonstrates how a Java application connects to an Oracle database using JDBC and performs basic banking transactions in a structured, layered architecture.

It includes:
- Account validation
- Balance check
- Money transfer between accounts
- Custom exception handling for insufficient funds
 Objectives

- To understand JDBC connectivity with Oracle Database  
- To implement DAO and Service layers in Java  
- To perform CRUD operations using `PreparedStatement`  
- To handle business logic with custom exceptions  
- To simulate real-world banking transactions  

Features

   Validate account numbers  
   Check account balance  
   Transfer money between two accounts  
   Handle insufficient funds using `InsufficientFundsException`  
   Uses Oracle sequences for transaction ID generation
